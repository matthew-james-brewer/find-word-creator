Create a Find Words Game

INSTRUCTIONS:

  • Enter a list of correct answers for the game in the "all correct answers" field. One answer per line.
  • Enter a name for the game in the "word" field.
  • Click the "Open in New Tab" button to preview the game in a new tab.
  • Click the "Share" button to get a link to share the game with others.
  • Click the "Download" button to download the game as an HTML file.

Instructions for playing the generated game:

  • Click the 'check' button to see if your answers are correct.
